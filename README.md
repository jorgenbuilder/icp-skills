# Project

This repository was created from the Agentspace template.

## Removing Agentspace Files

Once you're done setting up your project and no longer need the agentspace template files, you can remove them:

```bash
# Remove agentspace documentation and setup files
rm -rf agentspace/

# Remove devcontainer configuration (if you don't need Codespaces)
rm -rf .devcontainer/
```

Note: The `.devcontainer/` directory is only needed if you're using GitHub Codespaces. If you're working locally or using a different development environment, you can remove it.
